TeachAI — Ready to run
======================

1. ADD YOUR API KEY (required for chat & PDF upload)
   Open .env in this folder and set:
   GOOGLE_API_KEY=your_key_from_https://aistudio.google.com/apikey

2. START THE APP
   Double-click or run in PowerShell:
   .\scripts\start.ps1

   Or manually:
   - Backend:  cd backend  →  .\.venv\Scripts\python.exe run.py
   - Frontend: cd frontend →  npm run dev

3. OPEN IN BROWSER
   http://localhost:3000  (or 3001 if 3000 is busy)

Backend API: http://localhost:8000/docs

Already installed: Python 3.12, Node.js, all pip & npm packages.
